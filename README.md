# nvim_config
mpu69's neovim configuration 

### Installation
Linux & MacOS
```
git clone https://github.com/mpu69/nvim_config.git "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim
```

Windows 

```
git clone https://github.com/mpu69/nvim_config.git %userprofile%\AppData\Local\nvim\ 
```
